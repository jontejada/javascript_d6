Logical Operators Example
expr1 && expr2 AND returns expr1 if it can be converted to false, else returns expr2 thus, returns true if both are true true && true //true
expr1 || expr2 OR returns expr1 if it can be converted to true, else returns expr2 thus, returns true if either is true and false if noth are false false || true //true
!expr NOT returns false if expr can be converted to true, else true !true //false

if control structure

if (condition) {code to execute if condition is true}

if else control structure

if (condition) {code to execute if condition is true} else {code to execute if condition is false}

else if control structure

if (condition1) {code to execute if condition1 is true} else if (condition2) {code to execute if condition1 is false and condition2 is true} else {code to execute if both conditions are false}

for control structure

for (execute before the loop; condition for the loop; execute after the loop) {loop code to execute}

while control structure

while (condition for loop) {loop code to execute}
